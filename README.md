###terminal  
**supported unix-like commands:**  
`whoami`  
`ls [-l] [-d] [--directory] [-a] [--all]`  
`cat [file ...]`  
`help`  
`clear`  
`sudo [command [arg ...]]`  
`reboot`  
`pwd`  

**shell programs:**  
`psh [--version]`  
`blog`  
`victionarius`  

**supported features*:**  
tab completion  
command history  
web url permalinks (shareable urls)  

*_advanced features like bash scripting, short option bundling, and globbing are not supported_
